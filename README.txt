Anti-FactoryTube v1.1
=======================
Features:
- Hide AI/industrial low-quality videos on YouTube using keyword matching or regular expressions.
- Popup UI to manage keywords and options.
- Import/export keyword & settings JSON.
- Undo recently hidden items (from popup).
- Right-click context menu on YouTube pages to add the current channel to blacklist.
- Syncs settings via chrome.storage.sync.

Install (developer mode):
1. Unzip folder.
2. Open chrome://extensions, enable Developer mode.
3. Load unpacked and select this folder.

Publishing:
- Prepare icons, screenshots, privacy policy.
- Package and upload to Chrome Web Store Developer Dashboard.
